# leave-management-system

### Folder Structure

```
/public        - Web-accessible entry files (index.php, pages)
/includes      - Shared PHP components (header.php, footer.php, config.php, auth.php)
/assets        - CSS, JS, and static assets
/database      - SQL schema and database files
```

### Key Features

- Employee leave request submission
- Admin approval and rejection of requests
- User management (add, edit, delete users)
- Leave policy management
- Dashboard for leave statistics
- Secure authentication and session management

This project is built using PHP and MySQL, and is intended for deployment on a local or remote web server.
